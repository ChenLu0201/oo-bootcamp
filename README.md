**停车小弟**

Given:有一辆车需要小弟去停，且现在一号停车场还没停满  
When:小弟停车  
Then:取到票，停车成功

Given:有一辆车需要小弟去停，且现在一号停车场停满了，二号库没停满  
When:小弟停车  
Then:取到票，停车成功且停在二号库  

Given:有一辆车需要小弟去停，且现在n-1号停车场停满了，n号库没停满  
When:小弟停车  
Then:取到票，停车成功且停在n号库  

Given:有一辆车需要小弟去取，停在了1号库  
When:给小弟票，小弟取车  
Then:成功取车  

Given:有一辆车需要小弟去取，停在了1号库  
When:不给小弟票  
Then:小弟不取
